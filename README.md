# Starbucks Landing Page

A clean, responsive landing page inspired by Starbucks, built with plain HTML, CSS, and a bit of JavaScript for small interactions.

![Landing Page Screenshot](https://raw.githubusercontent.com/BinaryVortex/Starbucks-Landing-Page/main/Screenshot%202024-11-13%20054431.png)

## Overview

This repository contains a single-page static site that recreates a Starbucks-like landing page layout. The focus is on semantic HTML, modern CSS layout (Flexbox and Grid), responsive breakpoints, and accessible markup.

## Features

- Responsive layout that adapts to mobile, tablet, and desktop
- Hero section with CTA
- Grid-based content sections
- Reusable CSS classes and variables for consistent styling
- Small JavaScript enhancements (if present) to improve UX

## Built With

- HTML5
- CSS3 (Flexbox / Grid, custom properties)
- JavaScript (vanilla)

## Getting Started

To view the project locally:

1. Clone the repository

   git clone https://github.com/BinaryVortex/Starbucks-Landing-Page.git

2. Open the project folder and open `index.html` in your browser, or use a local server (recommended):

   - Using VS Code: Install the Live Server extension and click "Go Live"
   - Using Python 3: `python -m http.server 8000` then open `http://localhost:8000`
